# CCharlie Car
Demo Project

### API (Root)
build with nestjs

### Web app (./app)
build with react


### How to run
copy `./app/.env.example` to `./app/.env`

copy `.env.docker` to `.env`

run `./single_host.sh`, u might need to modify access with 'chmod'


### Docs for api
go to http://localhost:5000/docs/ or open swagger.json
